Команды Git
Команда Описание
git init Создайте пустой репозиторий Git или повторно инициализируйте существующий
git clone url Клонировать репозиторий в новый каталог
git add file Добавление отдельного файла
git add . Добавление всех файлов
git commit -m "comment" Записывайте изменения в хранилище, с комментарием к нему
git commit -a -m "comment" Запись всех файлов, исключая игнорируемые, с комментарием к ним
git push url Отправить коммиты в удаленный репозиторий
git status Список файлов без коммитов
git log Показывает журнал коммитов
git branch branch_name Добавление новой ветви
git checkout branch_name Переключиться на нужную ветку с именем branch_name
git checkout -b branch_name Создать ветку branch_name и переключиться на нее
git remote Список связанных с локальным удаленных репозиториев
